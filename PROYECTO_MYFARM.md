# Propuesta de Proyecto Profesional: MyPharm

## 1. Análisis del Producto (MVP)
**MyPharm** es una enciclopedia farmacéutica digital diseñada para ofrecer información técnica precisa y accesible. El proyecto destaca por:

*   **Arquitectura Escalable:** Separación de capas para un mantenimiento sencillo.
*   **Gestión Dinámica:** Carga de imágenes desde repositorios remotos para optimizar el peso de la app.
*   **Experiencia de Usuario (UX):** Buscador con sugerencias en tiempo real y navegación intuitiva.
*   **Identidad Visual:** Implementación de la paleta de colores ANPA (Azules y Turquesas) para transmitir seriedad y confianza médica.

## 2. Estimación de Costos de Desarrollo
El proyecto se ha valorado bajo un modelo de "Costo por Proyecto Cerrado" para un prototipo funcional (MVP).

*   **Horas de desarrollo efectivo:** 50 horas.
*   **Valor hora (Tarifa Junior/Freelance):** $10.000 CLP.
*   **Costo Total de Desarrollo:** **$500.000 CLP.**

## 3. Planificación y Cronograma
Para la entrega de una aplicación completa con las 20 categorías funcionales, se estima el siguiente cronograma:

*   **Dedicación diaria:** 2 horas.
*   **Tiempo total de ejecución:** **5 semanas.**

### Hitos clave:
*   **Semana 1:** Arquitectura y Navegación.
*   **Semana 2:** UI (Colores ANPA) y Buscador.
*   **Semana 3:** Ficha Técnica e integración de imágenes (Coil).
*   **Semana 4:** Carga masiva de base de datos (Antibióticos y otros).
*   **Semana 5:** QA, depuración y entrega de APK.

## 4. Recomendación Estratégica
MyPharm está diseñada para ser escalable. Su arquitectura permite integrar a futuro módulos de geolocalización de farmacias o sistemas de recordatorio de dosis (alarmas), aumentando su valor comercial.
